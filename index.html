Automator Customizer Script

This script controls AI Agents visibility based on SaaS plan status by checking your API and hiding premium features appropriately.

Quick Start

Add this script tag to any page where you want to control AI Agents visibility:

<script src="https://YOUR-USERNAME.github.io/automator-custom/automator-customizer.min.js"></script>

Replace YOUR-USERNAME and YOUR-REPO-NAME with your actual GitHub details.

How It Works

  • Automatically extracts location_id from URLs
  • Makes API calls to check SaaS plan status
  • Hides AI Agents for premium plans (Professional, Ultimate, Premium)
  • Shows AI Agents only for basic plans
  • Handles errors securely by hiding features

Files Available

  • automator-customizer.js - Full readable version
  • automator-customizer.min.js - Minified version for embedding

Setup Instructions

  1. Fork or download these files to your GitHub repository
  2. Enable GitHub Pages in your repository settings
  3. Update the script URL with your GitHub username and repository name
  4. Add the script tag to your target pages
